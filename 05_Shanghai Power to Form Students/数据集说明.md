# 数据集说明 —— 怎么自己建缓存(Data Collection 部分)

这本工作坊**默认离线可跑**:随包已带 3 站缓存(`data/<slug>/buildings.parquet`),
不下载任何东西、不联网,就能跑完 `05_完整流程` 的主流程。

但如果你想**亲手做 Data Collection**——下载「上海城市数据集」、从原始多源数据裁切+拼接、
建出自己的缓存(也才能换成另外的街道)——照这份说明走。配套 `05_完整流程` 的阶段 A。

> 原始数据集**体量大、商业授权,不随本工作坊分发**;请通过课程提供的渠道自行获取。

---

## 一、下载并解压

把「上海城市数据集」下载、解压到任意位置(例:`~/Downloads/上海城市数据集`)。
解压后里面是按编号分的文件夹(`01-建筑轮廓`、`02-POI&AOI`、`03-行政区`、`09-开源土地利用` …)。

## 二、本工作坊只用到 5 个文件(spine)

| 角色 | 相对数据集根的路径 |
|---|---|
| 街道边界(范围) | `03-行政区/2其他/乡镇街道/上海市_乡镇边界.shp` |
| AI建筑-带高度(几何+实测高度) | `01-建筑轮廓/②AI解析/上海市_建筑-AI影像解译-带高度.shp` |
| EULUC 开源土地利用(离散主键) | `09-开源土地利用/09-开源土地利用/开源建设用地分类/Data/上海市-开源建设用地.shp` |
| 带年份 Function(补充用途) | `01-建筑轮廓/③其它/上海市_建筑-带年份-内测版.shp` |
| 百度 AOI(辅助) | `02-POI&AOI/2-AOI/AOI-baiduapi/SHP/上海市_AOI.shp` |

> 每个 `.shp` 都要带齐它的同名伴随文件(`.shx` / `.dbf` / `.prj` 等),否则读不出。

## 三、在 config.py 填路径

打开 `config.py`,把 `DATASET_ROOT` 改成你解压的根目录(注意是上面 5 个路径的**公共上级**):

```python
DATASET_ROOT = "/Users/you/Downloads/上海城市数据集"   # ← 改成你的真实路径
SLUG = "lujiazui"                                       # 想建哪个站点
```

## 四、建缓存(三选一)

```bash
# 方式 A:命令行,建当前 SLUG
python3 engine/common.py lujiazui

# 方式 B:notebook —— 打开 05_完整流程,设好 DATASET_ROOT 后 Run All(阶段 A 的 build_or_load 会自动现建)

# 方式 C:Python
#   import sys; sys.path.insert(0, "engine")
#   import config, common as C
#   C.build_cache(config.site_name("caoyang"), "caoyang")
```

它会对该街道做**裁切 + 多源空间 join**,写出 `data/<slug>/buildings.parquet` 与 `site.yaml`。
之后这个站点就和随包缓存一样,完全离线可跑。

## 五、换成别的街道

> **换街道两招:①** 改 `config.SLUG` 在随包 9 个街道间离线切;**②** 按乡镇街道精确名从数据集现建任意新街道。这里只留要点。

`config.py` 的 `SITES` 表加一行(`name` 用乡镇街道层的精确名,可用「包含匹配」),
把 `SLUG` 指过去,再跑第四步建缓存即可。例:

```python
SITES = [
    {"slug": "tianzifang", "name": "打浦桥街道", "family": "居民/里弄+田子坊"},
    ...
]
```

## 常见问题

- **不下数据集行不行?** 行。随包 3 站缓存足够跑全部主流程;数据集只在你想**自己建缓存 / 换街道**时才需要。
- **找不到文件 `…AI影像解译-带高度.shp`?** 检查 `DATASET_ROOT` 是不是指到了 5 个路径的公共上级;
  跑「01」notebook 的第一格会逐个打印路径与「有/无」,照着核对。
- **建缓存很慢?** 第一次裁切大 shapefile + 多源 join 需要点时间(几十秒~几分钟,看街道大小),
  建好后读 parquet 是秒级,只需建一次。
- **缺套件?** `pip install -r engine/requirements.txt`(建缓存需 geopandas + pyarrow;卫星底图需 contextily)。
